MediaForge - Advanced Media Converter & Compressor
===================================================

Thank you for installing MediaForge!

Features:
- Convert images between PNG, JPG, WebP, AVIF, JPEG XL, HEIC, and more
- Convert videos between MP4, MKV, WebM with AV1, H.265, H.264 codecs
- Lossless compression to reduce file sizes without quality loss
- GPU acceleration with NVIDIA NVENC for faster processing
- Batch processing for multiple files
- Drag & drop support
- Dark and light themes

Getting Started:
1. Launch MediaForge from the Start Menu or Desktop shortcut
2. Drag files or folders onto the main window
3. Select your desired output format and compression settings
4. Click "Start" to begin conversion

GPU Acceleration:
If you have an NVIDIA GPU (GTX 600 series or newer), MediaForge will 
automatically use hardware acceleration for video encoding. This can 
significantly speed up video conversion.
